# DN5 DotNet FSE Angular Deep Skilling CTS

This repository contains the `Upskilling` folder with the following modules:

- `Module1_HTML5`
- `Module1_CSS3`
- `Module1_Bootstrap5`
- `Module1_JavaScript`
- `Module1_jQuery`
- `Module2_ANSI SQL Using MySQL`
- `Module3_C#ADO.NET`

## How to run

- For HTML/CSS/JS modules: open the `index.html` files in the browser.
- For SQL module: use MySQL and run the `.sql` scripts in `Module2_ANSI SQL Using MySQL`.
- For C# ADO.NET module: use .NET SDK and run `dotnet run --project "Upskilling/Module3_C#ADO.NET/Module3_CSharp_ADO.csproj"`.
